# Data Directory

All data files in this directory are **gitignored**. This document describes each data source and how to obtain it.

## Directory Structure

```
data/
├── raw/                     # Downloaded source data (not modified)
│   ├── ameriflux/           # AmeriFlux tower data (Ha1, Ha2, xHA)
│   ├── NEON_eddy-flux/      # NEON eddy covariance (DP4.00200.001)
│   ├── 7810_Processed/      # LI-7810 tree flux raw data
│   ├── phenocam/            # PhenoCam vegetation indices
│   ├── upland_wetland/      # Site classification data
│   └── ...
├── input/                   # Curated input files used by analysis scripts
│   ├── HF_2023-2025_tree_flux.csv
│   ├── HF_2023-2025_tree_flux_corrected.csv
│   ├── hummock_hollow.csv
│   ├── tomography_results_compiled.csv
│   └── tomography/          # ERT and sonic tomography images
└── processed/               # Generated by scripts (intermediate outputs)
    ├── aligned_hourly_dataset.csv
    ├── neon_swc_hourly.csv
    ├── tower_swc_ts_hourly.csv
    ├── wtd_met.csv
    └── ...
```

## Data Sources

### Raw Data (`data/raw/`)

| Folder | Source | Description | How to Obtain |
|--------|--------|-------------|---------------|
| `ameriflux/` | [AmeriFlux](https://ameriflux.lbl.gov/) | Tower flux data for US-Ha1, US-Ha2, US-xHA | Download BASE products from AmeriFlux website |
| `NEON_eddy-flux/` | [NEON](https://data.neonscience.org/) | Eddy covariance data (DP4.00200.001) for HARV | Run `scripts/01_import/04_neon_download.R` or download from NEON portal |
| `7810_Processed/` | Field measurements | LI-7810 tree flux chamber measurements | Contact authors |
| `phenocam/` | [PhenoCam](https://phenocam.nau.edu/) | GCC and NDVI from harvardems2 camera | Download from PhenoCam Network |
| `upland_wetland/` | Harvard Forest | Site classification data | Contact authors |

### Input Data (`data/input/`)

| File | Description | Generated By |
|------|-------------|--------------|
| `HF_2023-2025_tree_flux.csv` | Raw tree CH4 flux measurements | Data processing pipeline (external) |
| `HF_2023-2025_tree_flux_corrected.csv` | Unit-corrected flux data (2023-24 values x1000) | `scripts/02_analysis/01_timeseries.R` |
| `hummock_hollow.csv` | Tree locations (hummock vs hollow) | Field measurements |
| `tomography_results_compiled.csv` | ERT and sonic tomography metrics | External analysis |
| `tomography/` | ERT and sonic tomography images | Field measurements |

### Processed Data (`data/processed/`)

These files are generated by the import scripts. Re-run Phase 1 to regenerate.

| File | Description | Generated By |
|------|-------------|--------------|
| `aligned_hourly_dataset.csv` | Master hourly environmental dataset | `scripts/01_import/08_align.R` |
| `neon_swc_hourly.csv` | NEON soil water content (hourly) | `scripts/01_import/06_preprocess_soil_moisture.R` |
| `tower_swc_ts_hourly.csv` | Tower SWC and soil temperature (hourly) | `scripts/01_import/06_preprocess_soil_moisture.R` |
| `wtd_met.csv` | Water table depth + Fisher Met data | `scripts/01_import/03_tower_moisture.R` / `04_join_met.R` |
